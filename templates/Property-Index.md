## How to Add a Property

1. Copy `templates/property-template.md` to `03_Property/{Property-Name}/README.md`
2. Fill in the property info, contacts, and systems sections
3. Add a row to this index table
4. Store related documents (deeds, surveys, inspections) in the property folder
